# Plot ladle dimension diagnostic

Plot ladle dimension diagnostic

## Usage

``` r
plot_ladle(ladle, ...)
```

## Arguments

- ladle:

  Object returned by select_dimension_ladle().

- ...:

  Additional graphical arguments passed to
  [`graphics::matplot()`](https://rdrr.io/r/graphics/matplot.html).

## Value

Invisibly returns ladle table.
