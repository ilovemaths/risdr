# Construct true central subspace basis

Construct true central subspace basis

## Usage

``` r
make_true_beta(
  p,
  d,
  beta_type = c("coordinate", "random_sparse", "random_dense")
)
```

## Arguments

- p:

  Number of predictors.

- d:

  Structural dimension.

- beta_type:

  Basis type.

## Value

A p x d orthonormal basis matrix.
