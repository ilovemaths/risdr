# Coefficient of determination

Coefficient of determination

## Usage

``` r
r_squared(y_true, y_pred)
```

## Arguments

- y_true:

  Observed response values.

- y_pred:

  Predicted response values.

## Value

Numeric R-squared.

## Examples

``` r
r_squared(c(2, 4, 6, 8), c(2.2, 3.8, 5.7, 8.1))
#> [1] 0.991
```
